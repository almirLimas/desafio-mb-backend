# Backend - Desafio MB

Este é o backend da aplicação desenvolvida para o **Desafio MB**, utilizando **Node.js** com **Express** para criação das rotas de API. Ele funciona em conjunto com o frontend construído em **Vue 3 + Vite**.

---

## Tecnologias Utilizadas

- [Node.js]
- [Express]
- JavaScript

---

## Pré-requisitos

Antes de rodar o projeto, certifique-se de ter instalado:

- [Node.js]
- [npm] (gerenciador de pacotes do Node.js)

---

## ▶ Como rodar o projeto

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/almirLimas/desafio-mb-backend.git
   ```

   2. execute node --watch server.js ou npm start
